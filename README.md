# Molecular Solubility Prediction using Machine Learning

## Problem Statement
Predict aqueous solubility of chemical compounds using regression models.

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## Results
Linear Regression achieved Testing R² = 0.789

## Technologies
Python, Pandas, NumPy, Scikit-Learn, Matplotlib

## Key Learning
- Model comparison
- Overfitting analysis
- Performance evaluation
